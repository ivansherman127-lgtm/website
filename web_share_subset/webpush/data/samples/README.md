# Sample Data

Format examples from the Bitrix analytics pipeline.

- **`raw_bitrix_deals_sample.json`** — 30 rows from `raw_bitrix_deals` (wide raw Bitrix export, uploaded to D1 as `raw_bitrix_deals_p01..p04`)
- **`mart_deals_enriched_sample.json`** — 30 rows from `mart_deals_enriched` (Bitrix deals with enrichment + revenue calculations)
- **`raw_source_batches_sample.json`** — Metadata rows tracking data ingestion batches

Use these to understand the schema and data structure when working with the analytics backend.
